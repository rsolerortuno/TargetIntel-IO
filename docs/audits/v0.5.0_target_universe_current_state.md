# v0.5.0 target-universe current-state audit

This audit is an architectural inventory, not biological validation.

The pre-existing internal benchmark is `configs/benchmark_targets.yaml`: 56
symbols in 16 groups. It includes checkpoint, antigen-presentation, IFN-gamma,
myeloid, metabolic, stromal, plasticity, tumour-intrinsic, immune-context,
Treg and context-control lists. Current controls include `context_controls` and
`poor_direct_targets`; existing group expectations are used by the old
benchmark evaluator and therefore must not enter the new discovery features.
`configs/resistance_axes.yaml` is the canonical current axis list. TAP1,
IFNGR1 and STAT1 occur in that configuration, benchmark configuration,
role/modality rules, examples, and feasibility fixtures.

The existing `targetintel.target_universe.augment_target_universe` is the
canonical legacy Open Targets augmentation surface: it normalizes and
deduplicates required symbols, preserves an Open Targets/required-symbol source
flag, and sorts display rows. It is intentionally not reused for this issue:
it lacks immutable entry provenance, universe type, leakage controls and a
DepMap background contract. The selected canonical Issue 504 surface is
`targetintel.functional_dependency.target_universes`, alongside the existing
Issue 502 `gene_index.tsv` contract.

Issue 502 target-subset identity is order independent; required symbols are
deduplicated. Legacy augmentation retains only coarse inclusion provenance.
Current benchmark labels can reach legacy feature/ranking workflows through
benchmark evaluation, so the new discovery constructor rejects them. Existing
universe construction has no DepMap-outcome input; Issue 503 profiles are
separate and are not imported by this layer.

The frozen context benchmark is
`contexts/melanoma_anti_pd1/benchmarks/benchmark_v1.tsv`: 56 entries converted
from the existing internal configuration, with six controlled classes and a
deterministic development/holdout partition. The frozen discovery-source
artifact is `contexts/melanoma_anti_pd1/universes/discovery_sources_v1.tsv`.
It contains 55 unique canonical seed genes from the existing resistance-axis
configuration; policy-required benchmark union produces a 66-gene discovery
universe. These artifacts retain source keys and make no use of DepMap values.

Gap: current reviewed pre-DepMap source records are still insufficient for the
300–1,000-gene advisory discovery range. Additional independently curated
sources are required; no genes are invented to meet a count.
