# TargetIntel-IO example reports

This directory contains static example outputs generated by TargetIntel-IO for
anti-PD-1-resistant melanoma target triage.

## Included examples

| Target | Main therapeutic interpretation | Example report |
| --- | --- | --- |
| CTLA4 | Antibody / IO-combination target | [Open report](html_reports/CTLA4.html) |
| B2M | Antigen-presentation resistance biomarker | [Open report](html_reports/B2M.html) |
| BRAF | Tumor-intrinsic small-molecule target | [Open report](html_reports/BRAF.html) |

The [HTML report index](html_reports/index.html) summarizes the top representative
candidate from each therapeutic-intent mode.

## Interpretation

TargetIntel-IO keeps the biological classification of each target stable while
changing its priority according to the therapeutic question:

- **Antibody / IO-combination mode**
- **Resistance biomarker mode**
- **Tumor-intrinsic / small-molecule mode**

These reports are static portfolio examples generated from the rule-based
TargetIntel-IO MVP. They are intended for transparent hypothesis generation and
do not represent clinical recommendations or validated therapeutic conclusions.

## Regenerating the examples

From the repository root:

```bash
python scripts/06_generate_html_reports.py \
  --page-size 100 \
  --max-pages 3 \
  --top-n-per-mode 1 \
  --outdir examples/html_reports
